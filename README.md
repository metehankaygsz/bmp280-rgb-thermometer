# bmp280-rgb-thermometer
Control your RGB led using BMP280 sensor and Arduino


| BMP280 | Arduino |
| ------------- | ------------- |
| VCC  | 3.3 V  |
| GND  | GND  |
| SCL  | Analog 5 (A5)  |
| SDA  | Analog 4 (A4)   |



| RGB Led  | Arduino |
| ------------- | ------------- |
| Red (R) | Digital 1  |
| Green (G)  |Digital 3  |
| Blue (B) | Digital 2  |
